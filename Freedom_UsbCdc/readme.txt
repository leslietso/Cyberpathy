==========

This project demonstrates USB CDC (Communication Device Class) with the FRDM-KL25Z Freedom board.
Once enumarated, the red RGB LED turns green, and strings sent to the virtual COM port are echoed back to the host.


See http://mcuoneclipse.wordpress.com/2012/09/30/tutorial-usb-cdc-with-the-kl25z-freedom-board/
